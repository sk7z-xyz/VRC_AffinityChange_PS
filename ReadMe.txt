※本ZipファイルはWindowsのエクスプローラーで解凍すると､セキュリティ保護がかかり実行できません｡

下記の手順で､MASK値､動作確認､スタートアップへの追加を行ってください｡

1.VRC_AffinityChange_PS.ps1を編集してMASK値を自分のCPUに適した値に設定してください｡(PowerShellのエディタは文字化けするので非推奨です｡)

2.START_VRC_AffinityChange_PS.batを実行後､VRChatを起動し
PowerShellのコンソール上で｢Succes AffinityChange｣と表示されることを確認してください｡
また､タスクマネージャーからVRChatのスレッド割当が正しいことも確認してください

3.batファイルのショートカットを右クリックメニューから作成､ショートカットファイルを下記のスタートアップフォルダにコピーすることで
Windows起動時に自動起動することが可能となります｡
C:\Users\%USERNAME%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

■その他
PowerShellのウィンドウを非表示にしたい場合は､batファイルを確認してコメントアウト(REM)してを切り替えてください｡
※非表示にした場合､実行プロセスが特定できないため､Windowsを終了しない限りプロセスを終了することができません
動作確認後非表示にしてください｡
